# Kotan プロジェクト テーブル定義書

本ドキュメントは、「Kotan」プロジェクトにおけるデータベース（PostgreSQL）のテーブル構成をまとめたものです。

## 1. ユーザー・基盤系

### users (ユーザー管理)
ユーザーの基本情報と、EC購入金額に基づくガチャコイン管理を行います。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | ユーザーID |
| `email` | VARCHAR(255) | UNIQUE, NOT NULL | ログイン用メールアドレス |
| `password` | VARCHAR(255) | NOT NULL | ハッシュ化されたパスワード |
| `display_name` | VARCHAR(100) | | 画面表示用の名前 |
| `points` | INTEGER | DEFAULT 0 | 買い物などで貯まるポイント |
| `total_spent` | INTEGER | DEFAULT 0 | 累計購入金額 |
| `converted_spent`| INTEGER | DEFAULT 0 | ガチャコイン変換済みの金額 |
| `gacha_coins` | INTEGER | DEFAULT 0 | 所持ガチャコイン数 |
| `created_at` | TIMESTAMP | DEFAULT NOW() | 登録日時 |

---

## 2. EC・商品マスタ系

### categories (商品カテゴリ)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | カテゴリID |
| `name` | VARCHAR(50) | NOT NULL | カテゴリ名 (例: 食品, 家具, 魚) |

### products (商品・アイテムマスタ)
EC商品と、メタバース内で使用するアイテム（家具・種・魚など）を共通管理します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 商品ID |
| `category_id` | INTEGER | REFERENCES categories | カテゴリID |
| `name` | VARCHAR(200) | NOT NULL | 商品・アイテム名 |
| `description` | TEXT | | 詳細説明 |
| `price` | INTEGER | NOT NULL | 価格（円/ポイント） |
| `image_url` | VARCHAR(255) | | 画像URL |
| `item_type` | VARCHAR(20) | | 種別 (FOOD, FURNITURE, PLANT, FISH) |
| `weight` | REAL | DEFAULT 0 | 魚の重さ (Aquariumイベント用) |
| `growth_days` | INTEGER | DEFAULT 0 | 成長に必要な日数 (Garden用) |
| `stock` | INTEGER | DEFAULT 0 | 在庫数 |

---

## 3. ユーザー所持・配置系

### inventory (ユーザー所持品)
ユーザーが「今、カバンの中に持っている」アイテムを管理します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 所持品ID |
| `user_id` | INTEGER | REFERENCES users | 所有ユーザー |
| `product_id` | INTEGER | REFERENCES products | 商品ID |
| `quantity` | INTEGER | DEFAULT 1 | 所持数量 |

### area_placements (エリア配置)
RoomやGardenなどに配置されたアイテムの情報を管理します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 配置ID |
| `user_id` | INTEGER | REFERENCES users | 配置ユーザー |
| `product_id` | INTEGER | REFERENCES products | アイテムID |
| `area_type` | VARCHAR(20) | | エリア (ROOM, GARDEN, FARM) |
| `pos_x` | REAL | | X座標 |
| `pos_y` | REAL | | Y座標 |
| `pos_z` | REAL | | Z座標 |
| `rot_y` | REAL | | Y軸回転 |

## 4. アクティビティ専用

### garden_states (庭の成長管理)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 状態ID |
| `user_id` | INTEGER | REFERENCES users | ユーザーID |

| `target_weight` | REAL | | 目標合計重さ |
| `current_weight` | REAL | | 現在の合計重さ |
| `end_at` | TIMESTAMP | | 終了日時 |

### aquarium_contributions (貢献度管理)
| カラム名 | データ型 | 制約 | 説明 |

| `user_id` | INTEGER | REFERENCES users | ユーザーID |

## 5. 注文・決済系

### orders (注文概要)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 注文ID |
| `user_id` | INTEGER | REFERENCES users | 注文者 |
| `total_price` | INTEGER | NOT NULL | 合計金額 |
| `ordered_at` | TIMESTAMP | DEFAULT NOW() | 注文日時 |

### order_items (注文明細)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 明細ID |
| `order_id` | INTEGER | REFERENCES orders | 注文ID |
| `product_id` | INTEGER | REFERENCES products | 商品ID |
| `quantity` | INTEGER | NOT NULL | 数量 |
| `price_at_purchase`| INTEGER | NOT NULL | 購入時の単価 |
