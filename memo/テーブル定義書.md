# Kotan プロジェクト テーブル定義書

本ドキュメントは、「Kotan」プロジェクトにおけるデータベース（PostgreSQL）のテーブル構成をまとめたものです。

## 1. ユーザー・基盤系

### users (ユーザー管理)
ユーザーの基本情報と、EC購入金額に基づくガチャコイン管理を行います。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | ユーザーID |
| `email` | VARCHAR(255) | UNIQUE, NOT NULL | ログイン用メールアドレス |
| `password` | VARCHAR(255) | NOT NULL | ハッシュ化されたパスワード |
| `display_name` | VARCHAR(100) | | 画面表示用の名前 |
| `total_spent` | INTEGER | DEFAULT 0 | 累計購入金額 |
| `converted_spent`| INTEGER | DEFAULT 0 | ガチャコイン変換済みの金額 |
| `created_at` | TIMESTAMP | DEFAULT NOW() | 登録日時 |

### coupons (クーポンマスタ)
利用可能なクーポンの種類を定義します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | クーポンID |
| `code` | VARCHAR(50) | UNIQUE, NOT NULL | クーポンコード |
| `name` | VARCHAR(100) | NOT NULL | クーポン名 |
| `discount_rate` | INTEGER | | 値引き率 (%) |
| `min_amount` | INTEGER | | 適用可能な最低注文金額 |

### user_coupons (ユーザー所持クーポン)
ユーザーが現在持っている、または使用済みのクーポンを管理します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 管理ID |
| `user_id` | INTEGER | REFERENCES users | ユーザーID |
| `coupon_id` | INTEGER | REFERENCES coupons| クーポンID |
| `is_used` | BOOLEAN | DEFAULT FALSE | 使用済みフラグ |
| `expire_at` | TIMESTAMP | | このクーポンの使用期限 |
| `acquired_at` | TIMESTAMP | DEFAULT NOW() | 取得日時 |

## 2. EC・商品マスタ系

### categories (商品カテゴリ)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | カテゴリID |
| `name` | VARCHAR(50) | NOT NULL | カテゴリ名 (農産物, 水産物, 畜産物, その他) |

### products (商品・アイテムマスタ)
EC商品と、メタバース内で使用するアイテム（報酬家具・種・魚・ガチャコインなど）を共通管理します。
**仕様メモ**: 
- ECで**「農産物」カテゴリ**の商品を購入すると、対応する種が `inventory` に付与されます。
- 家具などの特殊アイテムは、**実績（achievements）**の達成報酬として `inventory` に付与されます。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 商品ID |
| `category_id` | INTEGER | REFERENCES categories | カテゴリID |
| `name` | VARCHAR(200) | NOT NULL | 商品・アイテム名 |
| `description` | TEXT | | 詳細説明 |
| `price` | INTEGER | NOT NULL | 価格（円） |
| `image_url` | VARCHAR(255) | | 画像URL |
| `item_type` | VARCHAR(20) | | 種別 (FOOD, FURNITURE, PLANT, FISH) |
| `weight` | REAL | DEFAULT 0 | 魚の重さ (Aquariumイベント用) |
| `growth_days` | INTEGER | DEFAULT 0 | 成長に必要な日数 (Garden用) |
| `stock` | INTEGER | DEFAULT 0 | 在庫数 |

---

## 3. ユーザー所持・配置系

### inventory (ユーザー所持品・種の手持ち)
ユーザーが「今、手元に持っている」すべてのアイテム（種、家具、魚、**ガチャコイン**など）を管理します。
庭に植える前の「種」や、獲得した「ガチャコイン」もここで個数を一括管理します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 所持品ID |
| `user_id` | INTEGER | REFERENCES users | 所有ユーザー |
| `product_id` | INTEGER | REFERENCES products | 商品・アイテムID |
| `quantity` | INTEGER | DEFAULT 1 | 所持数量 |

### room_placements (ルーム配置)
ルーム（部屋）内に配置された家具などのアイテム情報を管理します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 配置ID |
| `user_id` | INTEGER | REFERENCES users | 配置ユーザー |
| `product_id` | INTEGER | REFERENCES products | アイテムID |
| `pos_x` | REAL | | X座標 |
| `pos_y` | REAL | | Y座標 |

## 5. 実績・報酬系

### achievements (実績マスタ)
達成すべき目標と、その報酬となるアイテムを定義します。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 実績ID |
| `title` | VARCHAR(100) | NOT NULL | 実績名 |
| `description` | TEXT | | 達成条件の説明 |
| `reward_product_id`| INTEGER | REFERENCES products | 報酬アイテム（家具など）のID |

## 6. 注文・決済系

### orders (注文概要)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 注文ID |
| `user_id` | INTEGER | REFERENCES users | 注文者 |
| `total_price` | INTEGER | NOT NULL | 合計金額（値引き後） |
| `ordered_at` | TIMESTAMP | DEFAULT NOW() | 注文日時 |

### applied_coupons (注文適用クーポン)
1回の注文に適用されたクーポンを記録します（複数利用対応）。

| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 管理ID |
| `order_id` | INTEGER | REFERENCES orders | 注文ID |
| `user_coupon_id` | INTEGER | REFERENCES user_coupons | 適用した所持クーポンID |

### order_items (注文明細)
| カラム名 | データ型 | 制約 | 説明 |
| :--- | :--- | :--- | :--- |
| `id` | SERIAL | PRIMARY KEY | 明細ID |
| `order_id` | INTEGER | REFERENCES orders | 注文ID |
| `product_id` | INTEGER | REFERENCES products | 商品ID |
| `quantity` | INTEGER | NOT NULL | 数量 |
| `price_at_purchase`| INTEGER | NOT NULL | 購入時の単価 |
